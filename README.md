install virtualenv python 3.5

install python packages:
```
pip install -r requirements.txt
```

enable jupyterlab:
```
jupyter serverextension enable --py jupyterlab --sys-prefix
```

start jupyterlab:
```
jupyter lab
```


For a user guide see example.ipynb
